Advanced Web Exercises

The following exercises are designed to test your ability to synthesize the concepts you've learned into a single, comprehensive AI prompt. For each exercise, your task is not to write the code yourself, but to write a detailed prompt for an AI that specifies all the required features, layouts, and behaviors. Your prompt should be clear and specific enough for the AI to generate a complete and functional page that meets all the requirements.

Advanced Exercise 1: The Artist Bio Page

Filename: exercises/advanced-bio.html
Concept: This exercise combines a responsive two-column layout, custom typography, and styled interactive elements to create a professional and engaging artist biography page. It requires you to specify layout, responsive behavior, and visual styling in a single, cohesive prompt.

Your Task:

Write a single, detailed prompt for an AI that will generate a complete, responsive "Artist Bio" page. Your prompt must instruct the AI to include the following requirements:

  • Overall Structure: The page must have a consistent <header> and <footer>.
  • Layout: The main content area must be a two-column layout on desktop screens.
    • The left column should contain the image located at exercises/images/sample.jpg.
    • The right column should contain a heading (e.g., "About the Artist") and several paragraphs of placeholder text.
  • Responsiveness: On smaller screens (e.g., mobile), the two-column layout must collapse into a single column, with the image appearing above the text.
  • Typography: The page must import and use two different fonts from Google Fonts: one for the heading and another for the body text.
  • Interactive Element: Below the bio text, include a styled call-to-action button (e.g., "View My Work") that links to a placeholder page. The button must have a clear hover effect.
  • Styling: All styling must be in an external CSS file (css/advanced-bio.css).
  • Commenting: Your prompt must enforce the "neurotically detailed comments" rule from the introductory exercises, instructing the AI to explain the purpose and reasoning behind every HTML tag and CSS rule.
  • Prompt Archiving: Your prompt must instruct the AI to include the full prompt it receives as a comment block at the top of the main HTML file.

Advanced Exercise 2: The Artwork Series Gallery

Filename: exercises/advanced-gallery.html
Concept: This exercise focuses on creating a visually appealing and interactive gallery to showcase a series of artworks. It requires you to combine a responsive CSS Grid layout with interactive hover effects and a modal lightbox for viewing images.

Your Task:

Write a single, detailed prompt for an AI that will generate a complete, responsive "Artwork Series" gallery page. Your prompt must instruct the AI to include the following requirements:

  • Overall Structure: The page must have a consistent <header> and <footer>, as well as a main heading that serves as the title of the art series.
  • Grid Layout: The gallery must display the six available images (gallery1.jpg through gallery6.jpg from the exercises/images/ folder) in a responsive grid.
    • On desktop screens, the grid should be three columns.
    • On tablet screens, it should be two columns.
    • On mobile screens, it should be a single column.
  • Hover Effect: When a user hovers over a gallery image, a semi-transparent overlay containing the placeholder title of the artwork must appear.
  • Lightbox/Modal: When a user clicks on any gallery image, a larger version of that image must open in a centered modal window with a dimmed background. The modal must be closable by either clicking a close button ("X") or by clicking on the background overlay.
  • Styling & Functionality: All styling must be in css/advanced-gallery.css, and all modal functionality must be handled by scripts/advanced-gallery.js.
  • Commenting: Your prompt must enforce the "neurotically detailed comments" rule, instructing the AI to explain the purpose and reasoning behind every HTML tag, CSS rule, and JavaScript function.
  • Prompt Archiving: Your prompt must instruct the AI to include the full prompt it receives as a comment block at the top of the main HTML file.

Advanced Exercise 3: The Single Work Detail Page

Filename: exercises/advanced-work.html
Concept: This exercise focuses on creating a detailed view for a single piece of artwork, incorporating an embedded 3D model. It requires you to specify a clean, multi-column layout for presenting both visual and textual information effectively.

Your Task:

Write a single, detailed prompt for an AI that will generate a complete, responsive "Single Work" detail page. Your prompt must instruct the AI to include the following requirements:

  • Overall Structure: The page must have a consistent <header> and <footer>.
  • Layout: The main content must be a two-column layout.
    • The left column (taking up ~60-70% of the width) must contain a functional <model-viewer> element displaying a placeholder .glb 3D model.
    • The right column must contain the artwork's metadata, clearly structured with labels: "Title," "Artist," "Year," "Medium," and a "Description" with a paragraph of placeholder text.
  • Typography: The metadata in the right column must use clear and readable typography to distinguish between labels and content.
  • Responsiveness: On smaller screens, the layout must stack vertically, with the <model-viewer> element appearing above the metadata text.
  • Styling: All styling must be in an external CSS file (css/advanced-work.css).
  • Commenting: Your prompt must enforce the "neurotically detailed comments" rule, instructing the AI to explain the purpose and reasoning behind every HTML tag and CSS rule.
  • Prompt Archiving: Your prompt must instruct the AI to include the full prompt it receives as a comment block at the top of the main HTML file.

Advanced Exercise 4: The Full Homepage Layout

Filename: exercises/advanced-homepage.html
Concept: This is a capstone exercise that requires you to combine multiple components and concepts from the entire course into a single, cohesive homepage. You will need to specify a sticky header, a hero section, multiple content sections, and responsive behavior for all elements.

Your Task:

Write a single, detailed prompt for an AI that will generate a complete, responsive homepage. Your prompt must instruct the AI to include all of the following requirements:

  • Sticky Navigation: A <header> that is fixed to the top of the screen as the user scrolls.
    • On desktop, it should display a logo on the left and navigation links on the right.
    • On mobile, the navigation links should collapse into a "hamburger" icon that toggles a dropdown or overlay menu when clicked.
  • Hero Section: A full-height hero section directly below the navigation bar. It must have a background image using the file at exercises/images/hero.jpg and contain a centered main heading and a call-to-action button.
  • Content Sections: At least two additional content sections below the hero. Examples could include:
    • A short "About Me" section with a brief text intro and a link to the full bio page.
    • A "Featured Work" section displaying a small, responsive grid of 3-4 images (using files like gallery1.jpg, gallery2.jpg, etc.) that link to their respective work pages.
  • Footer: A <footer> at the bottom of the page containing social media links and a copyright notice.
  • Interactivity (Optional but Encouraged): Specify the use of subtle scroll-reveal animations to have content sections fade in as they enter the viewport.
  • Styling & Functionality: All styling must be in css/advanced-homepage.css, and all interactive functionality (like the mobile menu) must be in scripts/advanced-homepage.js.
  • Commenting: Your prompt must enforce the "neurotically detailed comments" rule, instructing the AI to explain the purpose and reasoning behind every HTML tag, CSS rule, and JavaScript function.
  • Prompt Archiving: Your prompt must instruct the AI to include the full prompt it receives as a comment block at the top of the main HTML file.
