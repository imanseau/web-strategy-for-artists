Web Concept Exercises

All of the following exercises will be created using AI prompts, not hand-coded. Each file should be saved inside a folder called exercises/. These pages are designed to isolate specific building blocks of modern web development; from layout and typography to navigation, interactivity, and responsiveness. Your goal is not just to make functional web pages, but to learn how to communicate effectively with an AI tool to produce clear, efficient, and well-structured code. Each exercise teaches you something new about web design, but also about prompt construction, the art of getting exactly what you need by describing it precisely. Pay close attention to the prompt before you run it, it’s a contract between you and the AI. Read it as if it were written for a freelance developer: are the instructions clear, complete, and specific? Does it describe not only what you want to appear on the page, but also how it should behave and why? After generating each file, study the resulting code carefully. The AI will include neurotically detailed comments explaining every line, use them. Read through the HTML, CSS, and JavaScript to understand how each piece fits together. Don't just glance at what shows up in the browser, examine the underlying structure and logic. This process will train you to see code not just as output, but as a readable, editable language, one you’ll be fluent in by the time you build your full portfolio site.

This section focuses on how web pages are structured using HTML5’s semantic elements, such as <header>, <section>, <nav>, and <footer>. You’ll explore how to organize content into logical blocks and use layout techniques like Flexbox and Grid to control spacing and alignment. Pay close attention to how the prompts describe page organization, are sections named clearly? Is spacing handled with CSS rather than brute force? Read through the AI-generated code and comments to understand the role each tag plays and how visual structure is achieved with external stylesheets. This section builds the foundation for everything that follows.
