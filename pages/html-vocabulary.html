Essential HTML Vocabulary

Understand the core concepts so you can describe what you want AI to build.

HTML

The basic language that structures web pages. Think of it as the bones of your site.

Element

A piece of content in a webpage, like a paragraph, image, or link. Most elements are in tag pairs like <p>...</p>.

Tag

The code that wraps around content to tell the browser what it is (like <h1>, <img>, etc.).

Attribute

Adds extra info to a tag (like src in <img src="..." /> or href in <a href="...">).

Heading (<h1>–<h6>)

Used for titles and section headers. Important for structure and SEO.

Paragraph (<p>)

A block of text. Most body content is placed in paragraph tags.

Link (<a>)

Creates a hyperlink. Essential for menus, buttons, and navigation.

Image (<img>)

Embeds pictures into the site. You'll use this to add portfolio images.

Div

A flexible container used to group content and control layout.

Class / ID

Labels you assign to elements so they can be styled or targeted.

List (<ul>, <ol>, <li>)

Used for menus, feature checklists, or ordered steps.

Form

Interactive fields (like contact or newsletter sign-ups). Often includes text boxes, dropdowns, and buttons.

Responsive Design

Means your site adjusts to different screen sizes (mobile, tablet, desktop).

Layout

How the sections of your site are arranged (e.g., header, body, footer, columns).

Semantic HTML

Uses meaningful tags like <header>, <footer>, <section> to describe content purpose.

Alt Text

A description for images (used for accessibility and SEO).

Favicon

The little icon that shows up in the browser tab.

Head / Title

Sets meta info like your page title (shows up on Google and browser tabs).

Iframe

Embeds content from another site (e.g., YouTube, Google Maps).

Charset

Sets the character encoding (almost always UTF-8 — just know it exists).

SEO (Search Engine Optimization)

The practice of making your website easier to find on search engines like Google. This includes using proper headings, alt text, titles, meta descriptions, and fast-loading content.
