Reference & Learning Resources

Use these to learn how websites work and improve your coding skills.

W3Schools – HTML

Learn HTML basics and test code live in your browser. Great for reference and experimentation.

  • https://www.w3schools.com/html/

W3Schools – CSS

Interactive tutorials and reference for CSS, how to style your website and control layout, color, fonts, and more.

  • https://www.w3schools.com/css/

W3Schools – JavaScript

JavaScript adds interactivity and dynamic features to your site. W3Schools is a good entry point to begin exploring scripts.

  • https://www.w3schools.com/js/
