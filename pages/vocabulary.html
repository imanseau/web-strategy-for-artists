Web Development Vocabulary

Core terms related to HTML and CSS structure and styling.

HTML Vocabulary

HTML
The basic language that structures web pages. Think of it as the bones of your site.
Element
A piece of content in a webpage, like a paragraph, image, or link. Most elements are in tag pairs like <p>...</p>.
Tag
The code that wraps around content to tell the browser what it is (like <h1>, <img>, etc.).
Attribute
Adds extra info to a tag (like src in <img src="..." /> or href in <a href="...">).
Heading (<h1>–<h6>)
Used for titles and section headers. Important for structure and SEO.
Paragraph (<p>)
A block of text. Most body content is placed in paragraph tags.
Link (<a>)
Creates a hyperlink. Essential for menus, buttons, and navigation.
Image (<img>)
Embeds pictures into the site. You'll use this to add portfolio images.
Div
A flexible container used to group content and control layout.
Class / ID
Labels you assign to elements so they can be styled or targeted.
List (<ul>, <ol>, <li>)
Used for menus, feature checklists, or ordered steps.
Form
Interactive fields (like contact or newsletter sign-ups). Often includes text boxes, dropdowns, and buttons.
Responsive Design
Means your site adjusts to different screen sizes (mobile, tablet, desktop).
Layout
How the sections of your site are arranged (e.g., header, body, footer, columns).
Semantic HTML
Uses meaningful tags like <header>, <footer>, <section> to describe content purpose.
Alt Text
A description for images (used for accessibility and SEO).
Favicon
The little icon that shows up in the browser tab.
Head / Title
Sets meta info like your page title (shows up on Google and browser tabs).
Iframe
Embeds content from another site (e.g., YouTube, Google Maps).
Charset
Sets the character encoding (almost always UTF-8 — just know it exists).
SEO (Search Engine Optimization)
The practice of making your website easier to find on search engines like Google. This includes using proper headings, alt text, titles, meta descriptions, and fast-loading content.

CSS Vocabulary

CSS
Cascading Style Sheets, the language used to style your HTML. It controls how everything looks (colors, fonts, spacing, etc.).
Selector
Targets specific elements to apply styles to (e.g., h1, .class, #id).
Property & Value
Style rules are written as property: value; like color: red; or font-size: 16px;.
Box Model
Everything in CSS is treated like a box: made up of content, padding, border, and margin.
Margin
The space outside an element, separates it from other elements.
Padding
The space inside an element, the gap between the content and the border.
Display
Controls how elements behave on the page. Common values: block, inline, flex, grid, none.
Position
Controls where an element appears on the page: static, relative, absolute, fixed, sticky.
Flexbox
A layout system for creating responsive row/column arrangements. Great for menus, grids, and centering.
Grid
Another layout system, useful for creating full-page or image gallery layouts with columns and rows.
Color
You can set text, background, border, and link colors using names (red), hex codes (#ff0000), or RGB values.
Font
Controls the typeface, size, weight (bold), and spacing of your text.
Background
Adds color, images, gradients, or patterns behind an element.
Link Styling
Change how links look using a, a:hover, etc. (color, underline, hover effects).
Responsive Design
Uses CSS to adjust layout and design for different screen sizes (mobile/tablet/desktop).
Media Query
A rule that applies styles based on screen size, key to responsive design.
Z-index
Controls which element sits on top when items overlap.
Border
Outlines an element; customizable by width, color, and style (solid, dashed, etc.).
Shadow
Add drop shadows or text shadows for emphasis or design effect.
Transition
Smoothly animates style changes (e.g., fade in/out on hover).
Class / ID
Used in both HTML and CSS to apply specific styles to elements.
Parallax Scrolling
A web design effect where background images move slower than foreground content as the user scrolls. It creates a sense of depth and can make a site feel more dynamic or immersive.
