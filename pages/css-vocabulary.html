Essential CSS Vocabulary

Understand how your siteâ€™s design and layout are controlled.

CSS

Cascading Style Sheets, the language used to style your HTML. It controls how everything looks (colors, fonts, spacing, etc.).

Selector

Targets specific elements to apply styles to (e.g., h1, .class, #id).

Property & Value

Style rules are written as property: value; like color: red; or font-size: 16px;.

Box Model

Everything in CSS is treated like a box: made up of content, padding, border, and margin.

Margin

The space outside an element, separates it from other elements.

Padding

The space inside an element, the gap between the content and the border.

Display

Controls how elements behave on the page. Common values: block, inline, flex, grid, none.

Position

Controls where an element appears on the page: static, relative, absolute, fixed, sticky.

Flexbox

A layout system for creating responsive row/column arrangements. Great for menus, grids, and centering.

Grid

Another layout system, useful for creating full-page or image gallery layouts with columns and rows.

Color

You can set text, background, border, and link colors using names (red), hex codes (#ff0000), or RGB values.

Font

Controls the typeface, size, weight (bold), and spacing of your text.

Background

Adds color, images, gradients, or patterns behind an element.

Link Styling

Change how links look using a, a:hover, etc. (color, underline, hover effects).

Responsive Design

Uses CSS to adjust layout and design for different screen sizes (mobile/tablet/desktop).

Media Query

A rule that applies styles based on screen size, key to responsive design.

Z-index

Controls which element sits on top when items overlap.

Border

Outlines an element; customizable by width, color, and style (solid, dashed, etc.).

Shadow

Add drop shadows or text shadows for emphasis or design effect.

Transition

Smoothly animates style changes (e.g., fade in/out on hover).

Class / ID

Used in both HTML and CSS to apply specific styles to elements.

Parallax Scrolling

A web design effect where background images move slower than foreground content as the user scrolls. It creates a sense of depth and can make a site feel more dynamic or immersive.
