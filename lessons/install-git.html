Install Git

This tutorial will guide you through the steps to install Git on your computer.

Step 1: Click "Launchpad"

Step 2: Search for "Terminal"

Step 3: Open "Terminal"

Step 4: Type "git" and press enter

Step 5: Click "Install"

Step 6: Click "Agree"

Step 7: Click "Done"

Step 8: Type "git --version" and press enter

Tip: If installed correctly, you will see something like "git version 2.39.5 (Apple Git-154)"
